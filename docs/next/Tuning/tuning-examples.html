Tuning examples and presets

caution

Please use the supplied defaults until a maiden flight has been conducted and the filtering is confirmed. Once this is done higher gains can be tested. Skipping this step and using these settings without adequate filtering could result in an uncontrollable helicopter.

How to use these presets?​

There are two options for using these pre-sets.

  1. Follow the screen shots to configure your helicopter.
  2. Load from the supplied Diff file. Please note: This method may overwright some of your previously configured settings. Please check and confirm carefully.

Creator:           Jonas Wackershauser @Johnny31297
Revision Date:       03.09.2024
Rotorflight Version:    2.0.0 Release
Rotorflight Hardware:   RadioMaster NEXUS
Tune Attributes:     Precise, Locked-In, 3D, Robotic

Hey there! As the creator of the OMPHOBBY M4 and avid Rotorflight 2 user, I want to share my personal Rotorflight 2 tune for my models with you. This is a highly tuned 3D setup for the M4, featuring great stability, excellent setpoint tracking and very precise and locked-in response. Be aware that this setup is specifically tuned for OMPHOBBY’s stock components, and that it might not perform optimally if you fly different components. You can, however, use it as a baseline for your own tune. As always, first verify that your filters are working correctly, then proceed onto the PIDFOB numbers.

This tune is set up to work with a universal transmitter profile. Channel order is AECR1T23.
Ensure all channels reach ±100% travel!
AUX1 for arming as per ELRS convention.
AUX2 for profile switching. ⚠ Mind the profile assignments, ensure proper AUX2 mapping ⚠
Profile #1 at -100% for 1850RPM
Profile #2 at -33% for 2500RPM
Profile #3 at +33% for 2850RPM
Profile #4 at +100% for Autorotation with bailout.

Enjoy!

Helicopter:     OMPHOBBY M4
Main Rotor Blades:   RotorTech 385mm Ultimate
Tail Rotor Blades: APEX 71mm
Swash Servos: OMPHOBBY DS2312C
Yaw Servo: OMPHOBBY DS2312T
Servo Arms: OMPHOBBY Plastic
FBL/Receiver: RadioMaster NEXUS & RP3-H
Motor: SUNNYSKY R80-5
ESC: OMPHOBBY 65A
Battery: OMPHOBBY 6S 2000mAh HV
Geometry: Stock 3D Geometry
ESC Setup: BEC 8V4, Governor Store, Governor mode M4
Tx Settings: ELRS 500Hz Wide
